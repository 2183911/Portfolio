Automatic Chess Board

An autonomous robotic chessboard allows human to play against Stocketfish chess engine on a physical chessboard. The chessboard employs Hall effect sensor matrix to detect magnetic chess pieces. The chessboard detects and communicates player's movement to a server running the Stockfish chess engine in real-time via WebSockets. According to Stockfish's move, the pieces are dragged by the magnet beath the board using CoreXy system.
